# CasaOS-UI

The web-gui of CasaOS, build with VueJS 2.7

## Project setup

```
yarn
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

Will be output to the ./public folder

### Lints and fixes files

```
yarn lint
```
